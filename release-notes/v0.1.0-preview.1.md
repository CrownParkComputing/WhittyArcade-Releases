# WhittyArcade v0.1.0 Preview 1

The first public WhittyArcade binary release for current CachyOS and Arch
Linux x86-64 systems.

## Highlights

- Board-grouped launcher for Namco System 22, Sega Model 1, Sega Model 2,
  Phoenix, Galaxian and Sega System 16B.
- ROM library supporting merged, split and non-merged MAME-compatible layouts.
- Global, per-board and per-game keyboard/controller profiles.
- OpenGL, Vulkan and software presentation backends.
- EEPROM/NVRAM management and verified high-score persistence.
- Sega Model 2 coverage includes Sega Rally Championship Revision C, with
  boot, video, geometry, I/O, audio and race-regression checks in the test
  suite.
