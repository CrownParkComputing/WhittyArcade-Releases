# WhittyArcade v0.2.0 Preview 2

This is the first WhittyArcade preview published for both Windows and Linux.
It provides one portable Windows 10/11 package and native archives for five
current Linux targets, all built from the same private engine commit.

## Desktop platforms

- Windows 10/11 x86-64 portable ZIP, including the required runtime DLLs.
- Ubuntu 24.04 x86-64.
- Debian 13 x86-64.
- Fedora 43 x86-64.
- openSUSE Tumbleweed x86-64.
- CachyOS / Arch Linux x86-64.
- Native file selection and per-user storage on both host families.
- No installer or administrator access is required on Windows.

## Emulator changes

- Adds Dirt Dash (World, DT2 Ver.C) and Aqua Jet (World, AJ2 Ver.B) to the
  supported Super System 22 catalog.
- Retains Time Crisis boot/gameplay, C352 audio, mouse light-gun aiming,
  trigger and pedal controls from the earlier Linux preview.
- Adds Galaxian and UniWar S to the playable classic-board catalog alongside
  Moon Cresta.
- Adds Virtua Cop 2 ROM-loading groundwork. Virtua Cop 2 is not yet exposed as
  a playable catalog entry in this release.
- Recognises 19 working sets. Ridge Racer Full Scale is also recognised but
  remains marked not working.
