# WhittyArcade v0.1.0 Preview 2

Preview 2 adds Time Crisis (World, TS2 Ver.B) and the Super System 22 hardware
path to the CachyOS / Arch Linux x86-64 release.

## Time Crisis

- Validated `timecris` ROM loading, including the 4 MiB program, texture,
  tilemap, point, sprite, sample and internal M37710 data regions.
- Super System 22 main bus, M37710 boot/input path, C374/VICS sprites,
  programmable gamma/fade, layer mixer, CZ/fog tables and point RAM.
- Mouse movement maps to calibrated absolute light-gun coordinates inside the
  4:3 game image.
- Left click fires.
- The player begins hidden. Hold `Space` to stand/attack; release `Space` or
  right-click to take cover and reload.
- A thick cyan P1 target sight replaces the desktop pointer. The shared gun
  interface also has a distinct red/magenta P2 sight for future two-player
  cabinet profiles.
- Existing per-game keyboard/controller profiles can replace the default gun,
  trigger and pedal inputs.
- The public Pages header now displays GitHub's anonymous count of Linux
  release-archive downloads without cookies or custom visitor analytics.

## ROM setup

Use a legally obtained World TS2 Ver.B `timecris.zip` and place
`namcoc71.zip` beside it, or import both through **ROM Library / Import**.
Time Crisis carries its M37710 firmware data and does not need
`namcoc74.zip`. Merged, split and non-merged MAME-compatible layouts remain
supported.

No ROMs, firmware, keys or links to game-data downloads are included or
provided.

## Preview status

Time Crisis boot, attract/gameplay scenes, audio and cabinet inputs are
working in this preview. Some Super System 22 fog and sprite alpha/blend
details remain experimental; this release does not claim pixel-perfect video.
